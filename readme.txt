# Following steps will help developers to generate the theme packages and upload them to the download site on Google Code automatically.
1. Download Apache Ant from http://ant.apache.org/bindownload.cgi.
2. Add the [ANT_HOME]/bin into the [PATH] of your computer.
3. Create *.modified file to zip and upload the specific theme to the Google Code.
4. cd to the project root directory in the console.
5. Run ant.
